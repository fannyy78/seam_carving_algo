L'objectif de ce package est de réduire la dimension d'image de façon automatique, en supprimant les pixels non essentiels à la photo (grâce à l'algorithme du Seam Carving).

Les codes sont disponibles en R et C++, en version algorithme naîf et dynamique, afin d'observer les différences de temps d'excéution et la performance. Le code en CPP dynamique étant le plus rapide et performant.

C'est à partir du fichier main que l'on fait tourner le code (en ajoutant une photo : depuis un package ou une source personnelle).
